============
Installation
============

At the command line::

    $ pip install powermolecli

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv powermolecli
    $ pip install powermolecli

Or, if you are using pipenv::

    $ pipenv install powermolecli

Or, if you are using pipx::

    $ pipx install powermolecli
